游戏过程会输出到log.txt中
每行一条信息，包括以下八种：
COIN r b 标志回合开始；此时红方有r个紫精灵，蓝方有b个紫精灵
SPAWN x 编号为x的结点上出现了一个紫精灵
PICK p x 编号为p的玩家（p=0或1，下同）获得了编号为x的结点上的紫精灵
MOVE p x y 编号为p的玩家从结点x移动到了结点y
LAB x y 玩家到达实验室且紫精灵足够，实验室中的选项依次为x和y
ADDMAJ p x 编号为p的玩家选择了实验室中的第x个选项（x=0或1）
RED t s 游戏已结束，红方听牌状态为t，战斗力为s
BLUE t s 游戏已结束，蓝方听牌状态为t，战斗力为s
注意到这八个单词的首字母都是不同的，便于判断信息类型。
